# 弹个幕小程序

竖屏灯牌 / 弹幕编辑器，可在微信开发者工具中直接导入运行。

## 功能

- 竖屏全屏预览，主文字自下而上滚动
- 场景预设（演唱会、酒吧、电竞、告白、音乐节）
- 字体、字号、速度、配色调节
- 本地保存样式、屏幕常亮
- 使用说明与隐私说明页

## 导入

1. 打开微信开发者工具 → 导入项目
2. 目录选择本仓库根目录（包含 `app.json` 的目录）
3. 本地预览可使用测试号；正式发布请在 `project.config.json` 中配置真实 `appid`

## 目录说明

| 路径 | 说明 |
|------|------|
| `pages/tanmu-editor/` | 主编辑器（入口页） |
| `pages/guide/` | 使用说明 |
| `pages/privacy/` | 隐私说明 |
| `tanmu-editor.html` | 网页原型参考，**不会**打入小程序包 |
